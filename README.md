# SEGA 32X

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **EXEREÚNESIS**.

SEGA 32X (32X) — a UD0 EXEREÚNESIS teardown + spec edition. A mushroom-shaped cap that plugged into the Genesis cartridge slot, the 32X strapped two Hitachi SH-2 processors onto the 16-bit machine as a stopgap before the Saturn. It split Sega's audience, confused buyers, and was abandoned within months — a cautionary tale of bridging hardware.
